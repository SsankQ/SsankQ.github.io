  #### ***토글 및 체크아웃 박스***

    ```markdown
    <details>
    <summary>토글 목록</summary>
    <div markdown="1">       
    - [ ]  체크아웃 1
    - [ ]  체크아웃 2
    </div>
    </details>
    ```

    <details>
    <summary>토글 목록</summary>

    - [ ]  체크아웃 1
    - [ ]  체크아웃 2

    </details>

  #### ***테이블*** 

    왼쪽 정렬 `|:---|`  
    가운데 정렬 `|:---:|`  
    오른쪽 정렬 `|---:|`  

    ```markdown
    |**영화 제목**|**주연 배우**|*장르*|
    |:---:|---:|---|
    |명량|최민식|역사|
    |인터스텔라|매튜 맥커너히|SF|
    |돈룩업|레오나르도 디카프리오|코미디, SF|
    ```

  #### ***프롬프트(Prompts)***

    ```markdown
    > `note` *타입 프롬프트*
    {: .prompt-note }

    > `warning` *타입 프롬프트*
    {: .prompt-warning }

    > `danger` *타입 프롬프트*
    {: .prompt-danger }
    ```


  > `note` *타입 프롬프트*
  {: .prompt-note }
